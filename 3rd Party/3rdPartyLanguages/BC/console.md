# Console

Very easy:
```
console("hi")
```
Output:
```
hi
```

Variable
```
a = "h"
console(a)
```
Output:
```
h
```

Join text
```
a = "h"
b = "i"
console(a, b, "!")
```
Output:
```
hi!
```
You can also use `+` instead of `,`.
